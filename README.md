ATM Simulation Program (Python)

This project is a simple ATM simulation system written in Python. It allows a user to enter a PIN and perform basic banking tasks such as checking balance, depositing money, and withdrawing funds.

Features

4-digit PIN verification

Up to 3 login attempts before the card is locked

Balance check option

Deposit money

Withdraw money with insufficient balance validation

Basic language selection (English supported)

How It Works

User selects a language (English only).

User enters a 4-digit PIN.

If the PIN is correct, the user can:

Check account balance

Deposit an amount

Withdraw an amount

If the wrong PIN is entered 3 times, the card gets locked.

Technologies Used

Python (Basic Input/Output, loops, conditions)

Purpose

This project is ideal for beginners learning Python and exploring basic program logic and user interaction.
